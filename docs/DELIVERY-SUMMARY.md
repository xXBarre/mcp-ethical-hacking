## Integration with Repository

The wiki should be placed in GitHub's wiki repository. Instructions:

1. Clone or create wiki repository:
   ```bash
   git clone https://github.com/tu-usuario/mcp-ethical-hacking.wiki.git
   cd mcp-ethical-hacking.wiki
   ```

2. Copy all .md files:
   ```bash
   cp /tmp/wiki/*.md .
   ```

3. Commit and push:
   ```bash
   git add .
   git commit -m "Add comprehensive wiki documentation"
   git push origin main
   ```

4. Verify on GitHub:
   - Visit: https://github.com/tu-usuario/mcp-ethical-hacking/wiki
   - Check sidebar appears
   - Verify all pages are linked
